

 ---------------------------------  Synchronous and Single thread -----------------------------

## 1. Synchronous means:
 { Synchronous means doing one thing at a time, One by one. Each task waits for the previous one to finish before starting.}

## Simple Definition: 
{ Synchronous = Step-by-step }

```javascript

console.log('B');
console.log('C');

// A
// B
// C
